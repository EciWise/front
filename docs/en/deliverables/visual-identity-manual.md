# Visual identity manual

## Identity idea

ECIWISE+ should feel institutional, human and intelligent. The interface must communicate academic trust, student support and responsible AI.

## Logo usage

Use `ECIWISE+` consistently. Keep the plus sign as part of the name. Do not distort, recolor arbitrarily, rotate or add decorative effects to the mark.

## Color palette

| Role | Value | Use |
| --- | --- | --- |
| Brand black | `#08080a` | Strong text and dark institutional surfaces |
| Brand red | `#c8102e` | Logo, primary actions and emphasis |
| Dark red | `#99000d` | Active and hover states |
| Soft red | `#e23a52` | Dark-theme accent |
| Admin purple | `#5a2a82` | Administrative accent |

## Typography

Use clean sans-serif typography. The product uses Inter/system fallbacks for general UI and Nunito-style rounded fields for controls.

Presentation guidance:

- Use short headings.
- Keep body text high contrast.
- Use screenshots over generic imagery.
- Use red for one key action or point per slide.

## Presentation structure

1. Problem.
2. Product promise.
3. User roles.
4. Feature walkthrough.
5. Architecture.
6. Adoption and impact.
7. Next steps.
